Input file

An input file has to be some kind of text file for example .txt, .doc, .... For MoSS to accept your inputs this file have to look a certain way it has
it has to be divided into an identification part, a value part and a
description of molecule part.

Identification
In this section you give all your input molecules an id
it could be a character, a number or why not a name.
The given identity is used in your idoutput file see Identifier output file.

Value
When you have a search where input data needs to be split into a focus and a complement part
this value is necessary. To be able to split the data into two parts a threshold How to understand an input and output file is set by default 0.5. If a value is greater then the value of threshold it is placed in the complement part
and if the value is less than the value of threshold it is put in the focus part.

Description
Here is the part where input molecules are described. Since MoSS only support
SMILES as a notation language your description needs to be written in this language.

Between the different parts you put a comma ",".

Example 1 - how an input file could look like:

id, value, description
a,0,CCS(O)(O)N
name,0,CCS(O)(C)N
1,0,CS(O)(C)N
2,0,CCS(=N)N
e,0,CS(=N)N
f,0,CS(=N)O

Output file

An output file will not only hold your mining result molecules but also other
features such as the number of nodes and frequency in percentage. To be more specific
the features displayed in the file are "id","description","nodes","edges","s_abs","s_rel","c_abs","c_rel".

"id" - To be able to separate the different results they are marked by an id number.
"description" - Returns a certain substructure that fulfill the search criteria written in SMILES.
"nodes" - Gives the number of nodes(atoms) encountered in the substructure.
"edges" - Gives the number of edges(bonds) encountered in the substructure.
"s_abs" - Absolute support of the substructure in the focus part

also known as the number of molecules in the focus part that contain this substructure.
"s_rel" - Relative support of the substructure in the focus part
the percentage of molecules in the focus part that contain this substructure.
"c_abs" - Absolute support of the substructure in the complement part
also known as the number of molecules in the complement part that contain this
substructure.
"c_rel" - Relative support of the substructure in the complement part
the percentage of molecules in the complement part that contain this substructure.

Example 2 - how an output file could look like bases on Example 1.

id,description,nodes,edges,s_abs,s_rel,c_abs,c_rel
1,O-S(-N)-C,4,3,3,50.0,0,0.0
2,O-S-C,3,2,4,66.666664,0,0.0
3,S(-N)-C-C,4,3,3,50.0,0,0.0
4,S(-N)-C,3,2,5,83.333336,0,0.0
5,S(-C)=N,3,2,3,50.0,0,0.0
6,S-C,2,1,6,100.0,0,0.0
7,N,1,0,6,100.0,0,0.0
